Interactive Map and Scatter Plot

Interactivity Instructions:

  • Click any point on the map or scatter plot. The selected point will change color to red and its size will double, indicating it's the selected point.
  • The corresponding point in the other plot will also increase in size and change color to yellow (a color not used in the price segmentation).
  • Clicking on a point in the scatter plot will zoom and center the map on the corresponding location, with animated effects to highlight the point through color and size changes.
  • Details for the selected rental unit, such as Address, Price, Sqft, Number of Bedrooms, Title, and URL, will be displayed in a table at the bottom of the page.
  • The previously selected point will return to its original size after a new selection is made.
